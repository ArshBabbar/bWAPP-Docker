WSDL stands for Web Service Description Language.
it is an XML-based file exposed by web services that informs client of the provided services/methods,including where they reside and the method-calling convertion.


dirb http://<target_ip>:port 

to find directory

Tool:ffuf

ffuf is a fast web fuzzer written in Go that allows typical directory discovery, virtual host discovery (without DNS records) and GET and POST parameter fuzzing.

Example:

ffuf -w /usr/share/seclists/Discovery/Web-Content/burp-parameter-names.txt -u 'http://<TARGET IP>:3002/wsdl?FUZZ' -fs 0 -mc 200
