Basic intro to Active Directory

Active Directory is a directory service in Windows network environment 
it is a distributed,hierarchical structure that allows for centeralized management of organizations resources(users,computerspolicies etc.)
it provides authentication and authorization function withing windows domain environment
it is a essential real-only database to all users withing the domain(all users can read AD )
