An object can be defined as any resource present within AD such as OU,computers,users etc.

Some Objects are :

Users -> are considered as leaf objects.
Contacts -> contains infromation like name,address etc,are considered leaf objects.
Printers
Computers
shared folders
Groups -> it is considered a container object.
Organizational Units(OU) 
Domains
Domain Controllers 
Sites ->A site in AD is a set of computers across one or more subnets connected using high-speed links. 
        They are used to make replication across domain controllers run efficiently.

Built in -> built in is a container that holds default groups in an AD domain.
Foreign Security Principles ->an FSP is an object created in AD to represent a security principal that belongs to a trusted external forest.
