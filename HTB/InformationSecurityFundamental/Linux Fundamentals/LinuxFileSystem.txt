Some Top Level directories are:

/ -> root,contaning all files required to boot the os before other filesystems are mounted, as well as files required to boot other filesystems.
/bin -> Contains essential command binaries
/boot -> Consists of static bootloader,kernel executable and files required to boot os
/dev -> Contains device files to facilitate access to every hardware device attached to the system
/etc -> Local system configuration files.configuration files for installed applications may be save here as well
/home -> each user on the system has a subDirectory here for storage
/lib -> shared library files that are required for system boot
/media -> external removable media devices such as USB devices are mounted here
/mnt -> Temporary mount point for regulare filesystems
/opt -> Optional files such as third-party tools can be saved here
/root -> the home directory of the root user
/sbin -> contains executables used for system administration(binary system files)
/tmp -> the operating system and many programs use this directory to store temporary file.generally cleared upon system boot and may be deleted at other times without warning
/usr -> contains executables,libraries,man files etc
/var -> contains variable data files such as log files,email in-boxes,web apllication related files, cron files and more
