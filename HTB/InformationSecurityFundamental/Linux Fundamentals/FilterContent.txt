Some Command that are used to filter content are :

More <file> --> it shows data in an interactive way we can scrool to see more and data stays on terminal

less <file> --> same as more but data dont stay on terminal.

head <file> --> head command is used to list top lines of the file,default top 10 lines.

tail <file> --> same as head but list from bottom,default bottom 10 lines.

sort <<usage<<  cat <file> | sort --> sorts the file according to alphabets 

grep --> grep command is used to find a particular thing in a file 

cut  <<usage<< cat <file> | grep -v "false\|nologin" | cut ":" -f1 --> used to cut a certain part of output here cut output after : .

Tr <<usage<< cat /etc/passwd | grep -v "false\|nologin" | tr ":" " " --> here : are replace by empty space tr is used to replace 

Column <<usage<< cat /etc/passwd | grep -v "false\|nologin" | tr ":" " " | column -t --> shows content in tabular form

Awk <<usage<< cat /etc/passwd | grep -v "false\|nologin" | tr ":" " " | awk '{print $1, $NF}' --> allows us to display the first ($1) and last ($NF) result of the line

Sed <<usage<<cat /etc/passwd | grep -v "false\|nologin" | tr ":" " " | awk '{print $1, $NF}' | sed 's/bin/HTB/g' -->it is used to substitute stuff we can use sed to replace part after s with part after /

Wc <<usage<< cat /etc/passwd | grep -v "false\|nologin" | tr ":" " " | awk '{print $1, $NF}' | wc -l --> wc -l is used to list total no of lines in output
